# gpui upstream trace

Tracks the pinned gpui revision vs the current `main` at
[zed-industries/zed](https://github.com/zed-industries/zed).
Maintained manually whenever a release lands or the maintainer
checks for upstream drift.

## Schema

| Field | Meaning |
|---|---|
| `Pinned` | Git rev pinned in Limn's `Cargo.toml` |
| `Upstream` | Latest `main` rev observed |
| `Delta` | Notable changes between pinned and upstream |
| `LastChecked` | ISO 8601 date |
