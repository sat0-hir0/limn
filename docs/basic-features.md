# エディタ基本機能 一覧（棚卸し）

> 「思考を深める」固有機能の前に、エディタとして最低限備えるべき基本機能の棚卸し。

## 「基本」と「本プロジェクト固有」の線引き

この一覧は**どのエディタにもある基本機能**。これらの上に、本プロジェクト固有の機能が乗る：

- ブロック構造（線形＋畳み）
- `/` コマンドによるブロック操作
- 打って化ける（Markdown 即時変換）
- カーソル直下の補完候補（3層司令塔）
- AI統合（選択→指示）
- リンク・バックリンク・グラフビュー
- フォーカスモード

固有機能に着手する前に、★★★ の基本機能が動くことが前提。
gpui-component の `editor` example は、1〜5 の ★★★ の多くを既に備えているので、そこを出発点にできる。

## 最初に固めるべき最小セット（★★★ 抜粋）

1. 文字入力＋IME＋改行
2. カーソル移動＋範囲選択＋全選択
3. 削除＋コピー/カット/ペースト＋Undo/Redo
4. `.md` を開く＋自動保存
5. スクロール＋カーソル追従
6. ウィンドウ管理＋ショートカット体系

この6つが動けば「テキストエディタ」として成立する。
固有機能はすべてこの上の階層。
